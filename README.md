# Дослідження чисельних методів вирішення нелінійних рівнянь

Цей репозиторій містить курсову роботу з дисципліни **«Методи комп'ютерних обчислень»** на тему: **«Дослідження чисельних методів вирішення нелінійних рівнянь»**.

## 📌 Опис проекту
Проект розв'язує нелінійне рівняння:
$$0.5x^5 - 0.005x - 1 = 0$$

Для пошуку коренів реалізовано такі методи:
1. **Метод половинного ділення (дихотомії)** (для дійсних коренів).
2. **Метод хорд** (для дійсних коренів).
3. **Метод дотичних (Ньютона)** (для дійсних коренів).
4. **Модифікований метод Ньютона** (для комплексних коренів).

Проект складається з двох основних частин:
1. **C# додаток (Windows Forms)** — графічний інтерфейс для інтерактивного обчислення коренів та виведення покрокових таблиць ітерацій.
2. **Python-скрипт** — автоматичний генератор повної пояснювальної записки (курсової роботи) у форматі `.docx` (Microsoft Word) відповідно до вимог ДСТУ 3008:2015, ДСТУ 8302:2015 та методичних вказівок ВНТУ.

---

## 📂 Структура проекту

* `MKO_LIB/` — вихідний код C# проекту (Windows Forms додаток на базі .NET 10).
  * `MKO_LIB/methods/` — реалізація чисельних методів (BisectionMethod, ChordMethod, NewtonMethod).
  * `MKO_LIB/proj/coursework.cs` — логіка виконання розрахунків та підготовки звіту в текстовому форматі для виведення в інтерфейсі.
  * `MKO_LIB/MainForm.cs` — графічний інтерфейс користувача.
  * `MKO_LIB.sln` — файл рішення (solution) Visual Studio.
* `generate_report.py` — Python-скрипт для генерації Word-звіту пояснювальної записки.
* `requirements.txt` — список Python-залежностей.
* `README.md` — цей посібник користувача.

## 🚀 Інструкція із запуску та команди

### 1. Робота з генератором звітів (Python)

Цей скрипт запускає чисельні методи, отримує таблиці ітерацій та генерує професійно оформлений файл `Пояснювальна_записка_МКО.docx`.

#### Встановлення залежностей
Для роботи скрипту необхідна бібліотека `python-docx`. Встановіть її за допомогою команди:
```bash
pip install -r requirements.txt
```
*Що робить ця команда:* Встановлює пакет `python-docx`, який дозволяє Python створювати та редагувати файли `.docx`.

#### Генерація звіту
Щоб згенерувати пояснювальну записку, виконайте команду:
```bash
python generate_report.py
```
*Що робить ця команда:*
- Проводить математичний розрахунок рівняння всіма 4 методами.
- Створює документ `Пояснювальна_записка_МКО.docx` згідно з ДСТУ (шрифт Times New Roman 14, полуторний інтервал, відступи полів 20мм/20мм/25мм/10мм).
- Автоматично генерує: Титульний аркуш, Індивідуальне завдання, Анотацію, Зміст, Вступ, Теоретичні відомості, Блок-схеми (якщо відповідні зображення `.png` присутні в каталозі), Таблиці ітерацій для кожного методу, Порівняльний аналіз швидкості збіжності, Висновки, Перелік посилань та Додатки з кодом.

---

### 2. Робота з графічним додатком (C# / Windows Forms)

Додаток надає зручний інтерфейс для інтерактивного обчислення та аналізу поведінки методів при зміні параметрів.

#### Компіляція проекту
Для збірки проекту виконайте команду:
```bash
dotnet build MKO_LIB\MKO_LIB.csproj
```
*Що робить ця команда:* Компілює C# проект, перевіряє код на помилки та створює виконуваний файл у теці `bin`.

#### Запуск додатку
Для запуску графічного інтерфейсу виконайте команду:
```bash
dotnet run --project MKO_LIB
```
*Що робить ця команда:* Компілює (при необхідності) та запускає додаток. Перед вами відкриється вікно програми "Equation Solvers (Coursework)".

**Алгоритм роботи в програмі:**
1. Введіть межі інтервалу пошуку дійсного кореня `a` та `b` (за замовчуванням `1.0` та `2.0`).
2. Вкажіть точність обчислень `eps` (за замовчуванням `0.01`).
3. Вкажіть початкове наближення для методу Ньютона `x0` (за замовчуванням `2.0`).
4. Натисніть кнопку **«Get»**:
   - Програма проведе обчислення всіма чотирма методами.
   - У текстовому полі з'явиться детальний звіт: покрокові таблиці ітерацій для методів половинного ділення, хорд, дотичних та комплексного методу Ньютона із зазначенням значень функцій, похибок та знайдених коренів.
5. Натисніть **«Clear»** для очищення результатів та введення нових значень.

## 📈 Параметри чисельних методів за замовчуванням
- **Функція**: $f(x) = 0.5x^5 - 0.005x - 1$
- **Інтервал ізоляції дійсного кореня**: $[1.0, 2.0]$
- **Точність**: $\epsilon = 0.01$
- **Початкове наближення для методу дотичних**: $x_0 = 2.0$
- **Початкові наближення для пошуку 4-х комплексних коренів**:
  - $z_1 = 0.5 + 1.0i$
  - $z_2 = 0.5 - 1.0i$
  - $z_3 = -1.0 + 0.5i$
  - $z_4 = -1.0 - 0.5i$
